# Machine Learning with Python

A collection of Jupyter notebooks covering the complete machine learning workflow using Python.
This repository includes data preprocessing, exploratory data analysis (EDA), implementation of machine learning algorithms, model training, evaluation, and performance comparison on real-world datasets.

## 📌 Repository Contents
- 📊 **Exploratory Data Analysis (EDA)**
  - Data cleaning
  - Missing value handling
  - Feature analysis
  - Correlation analysis
  - Data visualization using Matplotlib and Seaborn

- 🤖 **Machine Learning Algorithms**
  - Linear Regression
  - Polynomial Regression
  - Logistic Regression
  - Softmax Regression
  - Decision Trees
  - Support Vector Machines (SVM)
  - K-Nearest Neighbors (KNN)
  - K-Means Clustering

- 📈 **Model Training & Evaluation**
  - Train/Test Split
  - Feature Scaling
  - Performance Metrics
  - Model Comparison

- 📚 **Theory & Notes**
  - Mathematical foundations
  - Algorithm explanations
  - Implementation walkthroughs
  - Python code examples
  - Practical applications

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
